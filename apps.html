Sai (Dice)
2d is for 2 dice. 3d6 is for 3 dice. 2d10 is for 2 dice 10 sided. "Add to Homescreen" to use like as native app.
2d は2つのサイコロです。3d6 は3つのサイコロです。2d10 は2つの10面サイコロです。"ホーム画面に追加"をするとアプリのようにつかえます。
Karuta (Card)
2c52 (Blackjack) draws 2 playing cards. 5c54 (Poker) draws 5 playing cards including jokers. "Add to Homescreen" to use like as native app.
2c52(ブラックジャック) はトランプを2まいひきます。5c54(ポーカー) はジョーカーありのトランプを5まいひきます。"ホーム画面に追加"をするとアプリのようにつかえます。
Goban (Board)
b1x2x8 (Reversi) / 24b2x2x8 (Checker) places or flips black and white pieces on the board. b2x7 (7x7) / b2x9 (9x9) / b2x13 (13x13) is the board for the Igo. "Add to Homescreen" to use like as native app.
b1x2x8(リバーシ)・24b2x2x8(チェッカー) はボードに黒と白のコマをおいたりひっくりかえしたりします。b2x7(七路盤)・b2x9(九路盤)・b2x13(十三路盤) は囲碁盤です。"ホーム画面に追加"をするとアプリのようにつかえます。

Kuku
The web app for practicing mental arithmetic by tapping. You can see the time it took for the 20 questions. kuku?t10 (Tashite 10) is addition / subtraction up to 10. kuku?t20 (Tashite 20) is addition / subtraction up to 20. kuku?x9 (Kuku) is multiplication up to 9x9. kuku?x19 (Indian Kuku) is multiplication from 11x1 to 19x19. "Add to Homescreen" to use like as native app.
タップであんざんのれんしゅうをするウェブアプリです。20もんおわると、かかったじかんがわかります。kuku?t10(たして10) は10までのたしざん/ひきざんです。kuku?t20(たして20) は20までのたしざん/ひきざんです。kuku?x9(九九) は9×9までのかけざんです。kuku?x19(インド式九九) は11×1から19×19までのかけざんです。"ホーム画面に追加"をするとアプリのようにつかえます。
