Sai (Dice)
Sai is the web app that rolls dice by tapping. 2d is for 2 dice. 3d6 is for 3 dice. d2 is for die 2 sided like as coin toss. 2d10 is for 2 dice 10 sided. "Add to Homescreen" to use like as native app.
Saiはタップでサイコロをふるウェブアプリです。2dは2つのサイコロです。3d6は3つのサイコロです。d2はコイントスのような2面サイコロです。2d10は2つの10面サイコロです。"ホーム画面に追加"をするとアプリのようにつかえます。
Karuta (Card)
Karuta is the web app that draws cards or places chips on the board by touching. 2c52 is the playing cards for the Blackjack. 5c54 is the playing cards with joker for the Poker. b2x8 is the board for Reversi / Checker. b2x5 / b2x7 / b2x9 is the board for the Igo (5x5 / 7x7 / 9x9). "Add to Homescreen" to use like as native app.
Karutaはタッチでカードをとったりチップをおいたりするウェブアプリです。2c52はブラックジャック用のトランプです。5c54はポーカー用のジョーカーありトランプです。b2x8はリバーシ盤・チェッカー盤です。b2x5・b2x7・b2x9はの囲碁盤(五路盤・七路盤・九路盤)です。"ホーム画面に追加"をするとアプリのようにつかえます。
Kuku
Kuku is the web app for practicing mental arithmetic by tapping. You can see the time it took for the 20 questions. t10 (Tashite 10) is addition / subtraction up to 10. t20 (Tashite 20) is addition / subtraction up to 20. x9 (Kuku) is multiplication up to 9x9. x19 (Indian Kuku) is multiplication from 11x1 to 19x19. "Add to Homescreen" to use like as native app.
Kukuはタップであんざんのれんしゅうをするウェブアプリです。20もんおわると、かかったじかんがわかります。t10(たして10)は10までのたしざん/ひきざんです。t20(たして20)は20までのたしざん/ひきざんです。x9(九九)は9×9までのかけざんです。x19(インド式九九)は11×1から19×19までのかけざんです。"ホーム画面に追加"をするとアプリのようにつかえます。
